## Getting Started

### 1. Install

```shell
npm install -D @takuma-ru/rlse
```

```shell
pnpm add -D @takuma-ru/rlse
```

### 2. Add script to package.json

```json
{
  "scripts": {
    "rlse": "rlse -n <name> -l <patch | minor | major | pre> -c <build command>"
  }
}
```

### 3. Run

```shell
npm run rlse
```
